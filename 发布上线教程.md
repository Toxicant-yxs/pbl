# 发布上线傻瓜教程

> 从"代码在自己电脑上跑"到"所有人都能用的小程序"，跟着做就行！
> 预计总耗时：**2-4 小时**（其中域名备案需 1-7 天，可以提前弄）

---

## 上线前你需要准备的东西

| 东西 | 费用 | 说明 |
|------|------|------|
| 云服务器 | 约 50-100 元/年 | 推荐阿里云/腾讯云学生机，学生价 9.5 元/月起 |

## 第一阶段：买服务器和域名（提前做，备案要等）

### 1.1 买云服务器
推荐阿里云或腾讯云，学生认证后超便宜：

**阿里云（推荐）：**
1. 打开 https://www.aliyun.com
2. 搜索"云服务器ECS"，或者直接搜"学生机"
3. 配置选择：
   - **镜像**：Ubuntu 22.04 64位（不要选 Windows，Linux 上线更简单）
   - **配置**：1核2G 足够（学生机标配）
   - **带宽**：选 1M-3M 都可以
   - **地域**：选离你近的（比如华东、华北）
4. 设置 root 密码（一定记牢！）
5. 付款，拿到服务器的**公网 IP**（比如 123.45.67.89）

**腾讯云（备选）：**
流程一样，打开 https://cloud.tencent.com 搜"云服务器 CVM"

### 1.2 买域名
1. 在阿里云/腾讯云搜"域名注册"
2. 输入你想要的域名，比如 `todo-app.xyz` 或 `nidetodo.com`
3. 加入购物车，付款
4. **立刻去备案！**（重要）
   - 阿里云：控制台搜索"ICP备案"
   - 腾讯云：控制台搜索"网站备案"
   - 按提示上传资料、人脸核验
   - 等待 1-7 天审核通过

### 1.3 注册微信小程序账号
1. 打开 https://mp.weixin.qq.com
2. 点"立即注册"→ 选"小程序"
3. 按流程填写信息（个人账号免费但不能支付，企业账号需 300 元认证费）
4. 注册成功后，在"开发"→"开发管理"→"开发设置"里：
   - 记录下你的 **AppID**（类似 wx1234567890abcdef）
   - 后面要用到

## 第二阶段：连接服务器，安装环境

### 2.1 用 SSH 连接服务器

**Windows 用户：**
1. 下载 FinalShell：http://www.hostbuf.com/ （免费中文SSH工具）
2. 安装后打开，点击文件夹图标→"新建连接"
3. 填写：
   - 名称：随便填（比如"我的待办服务器"）
   - 主机：你的服务器公网 IP
   - 端口：22
   - 用户名：root
   - 密码：你买服务器时设置的密码
4. 双击连接，第一次会弹提示选"接受并保存"

**Mac 用户：**
打开终端，输入：
```bash
ssh root@你的服务器IP
# 输入密码（输入时不会显示*，正常的）
```

连接成功后你会看到类似 `root@xxx:~#` 的提示符，这就是你的服务器命令行了。

### 2.2 在服务器上安装 Node.js

复制粘贴下面的命令，一行一行执行（每行回车后等跑完再复制下一行）：

```bash
# 更新软件源
apt update && apt upgrade -y

# 安装 Node.js 18.x
curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
apt install -y nodejs

# 验证安装
node -v
npm -v
```

看到版本号就是成功了 ✅

### 2.3 安装 MySQL 8.0

```bash
# 安装 MySQL
apt install -y mysql-server

# 启动 MySQL 并设置开机自启
systemctl start mysql
systemctl enable mysql

# 设置 root 密码（把 your_password 改成你自己的密码，记牢！）
mysql -u root -e "ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'your_password'; FLUSH PRIVILEGES;"

# 验证
mysql -u root -p -e "SELECT VERSION();"
# 输入你刚设的密码，看到版本号就成功了
```

### 2.4 安装 Nginx（用于 HTTPS 和反向代理）

```bash
apt install -y nginx
systemctl start nginx
systemctl enable nginx
```

验证：浏览器地址栏输入 `http://你的服务器IP`，看到"Welcome to nginx!"页面就成功了 ✅

### 2.5 安装 PM2（让后端服务一直运行）

```bash
npm install -g pm2
pm2 -v
```

## 第三阶段：上传代码到服务器

### 3.1 上传项目代码

**最简单的方法：用 Git（推荐）**
在服务器上执行：
```bash
# 进入你喜欢的目录
cd /opt

# 克隆你的代码仓库
git clone <你的仓库地址> todo-app

# 如果是私有仓库，需要配置 SSH 密钥或者用 https + token
cd todo-app
ls
# 你应该能看到 01_启动、04_阶段三_前端开发、05_阶段三_后端开发 这些文件夹
```

**如果不会用 Git：**
- Windows 用 FinalShell：连接后左边有文件管理器，直接把本地项目文件夹拖到 `/opt/todo-app/` 下
- Mac 用 scp 命令：`scp -r /本地项目路径 root@服务器IP:/opt/todo-app/`

### 3.2 初始化服务器上的数据库

```bash
cd /opt/todo-app/05_阶段三_后端开发/server

# 导入数据库（your_password 是你 2.3 步设的 MySQL 密码）
mysql -u root -pyour_password < sql/schema.sql
```

### 3.3 配置后端环境变量

```bash
cd /opt/todo-app/05_阶段三_后端开发/server

# 复制配置文件
cp .env.example .env

# 编辑配置（安装 vim 如果没有）
apt install -y vim
vim .env
```

按 `i` 键进入编辑模式，修改以下内容：
```env
PORT=3000

DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASS=your_password        ← 改成你 2.3 步设的 MySQL 密码
DB_NAME=todo_app

JWT_SECRET=改成一个复杂点的随机字符串      ← 随便改一个长字符串，比如 todo-jwt-2024-xxxxx-super-secret

WX_APPID=你的微信小程序AppID              ← 填你 1.3 步拿到的 AppID
WX_SECRET=你的微信小程序AppSecret          ← 在微信公众平台"开发设置"里也能找到
```

编辑完按 `Esc`，然后输入 `:wq` 回车保存。

### 3.4 安装后端依赖并启动

```bash
cd /opt/todo-app/05_阶段三_后端开发/server

# 配置 npm 淘宝镜像（国内服务器更快）
npm config set registry https://registry.npmmirror.com

# 安装依赖
npm install --production

# 用 PM2 启动（让服务一直在后台跑）
pm2 start app.js --name todo-server

# 设置 PM2 开机自启
pm2 save
pm2 startup
# 它会输出一条命令，复制执行一下就行
```

验证后端是否跑起来了：
```bash
curl http://localhost:3000/health
# 应该返回 {"status":"ok",...}
```

## 第四阶段：配置域名和 HTTPS（微信小程序要求必须 HTTPS）

### 4.1 解析域名到服务器 IP

1. 登录你买域名的服务商控制台（阿里云/腾讯云）
2. 找到"域名解析"→"添加记录"
3. 添加两条：
   - 记录类型：`A`，主机记录：`api`，记录值：你的服务器公网 IP → 这样 `api.你的域名.com` 就指向服务器了
   - 记录类型：`A`，主机记录：`@`，记录值：你的服务器公网 IP → 根域名也指向服务器
4. 等待几分钟生效，可以在本地电脑 cmd 里 ping 一下：
   ```
   ping api.你的域名.com
   ```
   显示你的服务器 IP 就是解析成功了。

### 4.2 配置 Nginx 反向代理

```bash
# 创建 Nginx 配置文件
vim /etc/nginx/sites-available/todo-app
```

按 `i` 进入编辑，粘贴下面内容（**把 `api.yourdomain.com` 改成你自己的域名**）：

```nginx
server {
    listen 80;
    server_name api.yourdomain.com;

    location / {
        proxy_pass http://127.0.0.1:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
    }
}
```

按 `Esc`，输入 `:wq` 保存。然后：
```bash
# 启用配置
ln -s /etc/nginx/sites-available/todo-app /etc/nginx/sites-enabled/

# 删除默认配置（可选）
rm /etc/nginx/sites-enabled/default

# 测试配置有没有错
nginx -t

# 重启 Nginx
systemctl restart nginx
```

此时访问 `http://api.你的域名.com/health` 应该能看到接口返回了。

### 4.3 申请免费 HTTPS 证书（Let's Encrypt）

微信小程序强制要求 HTTPS，用免费的 Let's Encrypt 就行：

```bash
# 安装 certbot
apt install -y certbot python3-certbot-nginx

# 申请证书（把 api.yourdomain.com 改成你的域名）
certbot --nginx -d api.yourdomain.com
```

执行时会问你几个问题：
- 输入邮箱（用于到期提醒）
- 同意协议：Y
- 要不要接收广告邮件：N
- 是否强制跳转 HTTPS：选 **2**（Redirect，自动把 http 跳转到 https，更安全）

完成后会显示 `Congratulations!`，证书就装好了！🎉

现在访问 `https://api.你的域名.com/health` 应该能看到返回结果了（浏览器地址栏有小锁头🔒）。

## 第五阶段：配置微信小程序后台

### 5.1 修改小程序代码里的后端地址

在你本地电脑上，用 VS Code 打开文件：
`04_阶段三_前端开发/miniprogram/app.js`

找到这一行：
```javascript
apiBase: 'http://localhost:3000/api',
```

改成你的线上域名：
```javascript
apiBase: 'https://api.你的域名.com/api',
```

同时修改 `project.config.json`，把 `appid` 改成你自己的 AppID：
```json
"appid": "wx1234567890abcdef",
```

保存。

### 5.2 在微信公众平台配置服务器域名

1. 登录 https://mp.weixin.qq.com
2. 进入"开发"→"开发管理"→"开发设置"
3. 找到"服务器域名"，点击"修改"
4. 在 **request 合法域名** 里填入：`https://api.你的域名.com`
   （注意必须是 https，不能带末尾斜杠）
5. 保存提交

### 5.3 上传小程序代码

1. 打开微信开发者工具，确保项目是最新的代码
2. 点击右上角 **"上传"** 按钮
3. 填写版本号（比如 1.0.0）和项目备注
4. 上传成功后，回到微信公众平台
5. 进入"管理"→"版本管理"
6. 你会看到刚上传的版本在"开发版本"里
7. 点击 **"提交审核"**
8. 按提示填写审核信息：
   - 功能页面：主要页面都选上
   - 功能说明：简单描述（比如"个人待办事项管理工具，支持添加、分类、提醒待办"）
   - 如果是个人账号，注意不能有未开放的类目内容
9. 提交后等待审核（通常几小时到 1 天）
10. 审核通过后，在"版本管理"里点击 **"全量发布"**

发布成功！🎉 所有人都可以在微信里搜索到你的小程序了！

## 常见问题

### Q1：HTTPS 证书过期了怎么办？
Let's Encrypt 证书 90 天过期，但 certbot 会自动续期，不用管。可以手动测试续期：
```bash
certbot renew --dry-run
```

### Q2：服务器重启后后端没起来？
检查 PM2 自启是否配置：
```bash
pm2 status
# 如果没有 todo-server，重新启动：
cd /opt/todo-app/05_阶段三_后端开发/server
pm2 start app.js --name todo-server
pm2 save
```

### Q3：微信小程序请求报错"不在以下 request 合法域名列表中"？
1. 确认在微信公众平台配置了域名
2. 确认域名是 HTTPS（有小锁头）
3. 确认没有端口号（https 默认 443 不用写）
4. 微信开发者工具里点"清缓存"→"全部清除"，重新编译

### Q4：数据库连接不上？
检查 `.env` 里的 `DB_PASS` 是否正确，MySQL 是否在运行：
```bash
systemctl status mysql
```

### Q5：个人账号 vs 企业账号区别？
- **个人账号**：免费，不能用微信支付、不能涉及商业交易类功能
- **企业账号**：300 元/年认证费，功能全，可支付
- 待办工具类应用个人账号完全够用

### Q6：想换更便宜/更强的服务器？
买新服务器后，按本文档第二三四阶段重新来一遍，然后把域名解析改到新 IP 即可。
