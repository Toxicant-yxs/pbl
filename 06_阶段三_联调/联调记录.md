## 3. 联调中遇到的问题与解决

### 问题 1：小程序请求跨域报错
- **现象**：wx.request 报 `request:fail url not in domain list`
- **原因**：未配置 request 合法域名
- **解决**：开发期勾选"不校验合法域名"；上线后需在小程序后台配置 `https://your-domain.com`

### 问题 2：Token 过期未自动跳转
- **现象**：401 响应时停留在当前页
- **解决**：在 `utils/request.js` 统一拦截 401，清除本地存储并跳转 login 页
  ```js
  if (res.statusCode === 401) {
    wx.removeStorageSync('token');
    wx.reLaunch({ url: '/pages/login/login' });
  }
  ```

### 问题 3：分页 total 计算错误
- **现象**：搜索后 total 仍为全部数量
- **原因**：count 查询未复用 where 条件
- **解决**：在 todo.js 中将 where 条件同时应用于 count 查询
  ```js
  const [countRows] = await pool.query(
    `SELECT COUNT(*) AS total FROM todos t WHERE ${whereSql}`, params
  );
  ```

### 问题 4：日期时区错位
- **现象**：截止时间 23:00 在数据库变 15:00
- **原因**：MySQL 存储 DATETIME 默认按服务器时区
- **解决**：API 统一返回 ISO 8601 字符串（带 +08:00），前端不做时区转换

### 问题 5：JSON 字段布尔值丢失
- **现象**：MySQL TINYINT(1) 返回数字 0/1，前端 v-if 失效
- **解决**：在 DAO 层将 is_completed / is_pinned 强制转为 boolean
