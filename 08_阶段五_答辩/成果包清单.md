# 成果包清单与打包说明

> 提交时间：第 17 周答辩结束后 24 小时内

> 命名：`24软工X班-第N小组-个人待办-小程序实训成果包.zip`

## 2. 打包命令

```bash
# 在 /workspace 目录下
cd /workspace

# 清理临时文件
rm -rf docx_extracted/

# 创建成果包
zip -r "24软工X班-第5小组-个人待办-小程序实训成果包.zip" \
  进度跟踪表.md \
  01_启动 \
  02_阶段一_选题分工与资料搜集 \
  03_阶段二_系统设计 \
  04_阶段三_前端开发 \
  05_阶段三_后端开发 \
  06_阶段三_联调 \
  07_阶段四_测试与优化 \
  08_阶段五_答辩 \
  09_个人总结
```
