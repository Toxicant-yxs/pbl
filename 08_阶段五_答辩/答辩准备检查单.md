## 演示准备
- [ ] 微信开发者工具打开项目，登录 demo 账号
- [ ] 准备演示数据：12 条待办，分布在 4 个分类
- [ ] 提前点击全部页面，确保无控制台报错
- [ ] 后端服务 `npm start` 已运行，`/health` 返回 ok

## 演示流程（8 分钟）
- [ ] 0:00 打开小程序 → 登录页（介绍品牌"待办 · TODO"）
- [ ] 0:30 登录 → 今日待办（演示 5/12 数据）
- [ ] 1:30 点击 + → 新建待办（演示优先级/分类/截止时间/提醒）
- [ ] 3:00 保存 → 回到首页（演示列表已更新）
- [ ] 3:30 点击完成圆圈 → 标记完成（演示乐观更新）
- [ ] 4:00 切到分类 Tab → 点击"学习" → 分类筛选
- [ ] 5:00 切到统计 Tab → 演示完成率环、4 指标、柱状图
- [ ] 6:00 切到个人中心 → 演示总待办/已完成/总完成率
- [ ] 7:00 总结收尾 → 说出技术栈和亮点
