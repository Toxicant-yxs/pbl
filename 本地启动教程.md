# 本地启动傻瓜教程

> 跟着做就能跑起来！每一步都有截图说明级别的文字指导，不需要你懂代码。

## 第一步：安装 Node.js

1. 打开 https://nodejs.org/
2. 点击左边那个绿色大按钮 **"LTS"**（推荐版本），下载安装包
3. 双击安装包，一路点 "Next"（下一步），遇到勾选框都勾上（特别是 "Add to PATH"）
4. 安装完成后，**重启你的电脑**（重要！）
5. 验证安装：按下键盘 `Win + R`，输入 `cmd` 回车，黑窗口里输入：
   ```
   node -v
   npm -v
   ```
   如果都显示出版本号（比如 v20.x.x），说明装好了 ✅

---

## 第二步：安装 MySQL

### Windows 用户
1. 打开 https://dev.mysql.com/downloads/mysql/
2. 选择操作系统 Windows，下载 **MySQL Installer for Windows**
3. 运行安装程序，选择 **"Developer Default"**（开发者默认），一路 Next
4. 设置 root 密码的时候，**请输入 `root`**（和项目配置一致，省得改来改去）
5. 一路 Next 直到安装完成，MySQL 会自动启动

### Mac 用户
1. 推荐用 Homebrew 安装：打开终端输入
   ```bash
   brew install mysql
   brew services start mysql
   ```
2. 设置 root 密码为 `root`：
   ```bash
   mysql_secure_installation
   ```
   按提示设置密码为 `root`

### 验证 MySQL
打开命令行（Windows 用 cmd，Mac 用终端），输入：
```
mysql -u root -proot
```
如果能看到 `mysql>` 提示符，说明装好了。输入 `exit` 退出。

## 第三步：下载/克隆项目代码

### 方法一：用 Git（推荐）
1. 安装 Git：https://git-scm.com/downloads 一路下一步
2. 找一个你放代码的文件夹，比如 `D:\projects`
3. 在文件夹空白处按住 Shift + 右键，选择"在此处打开 PowerShell 窗口"
4. 输入：
   ```bash
   git clone <你的仓库地址>
   cd pbl
   ```

### 方法二：直接下载 ZIP
1. 打开你的代码仓库网页（GitHub/Gitee）
2. 点击绿色按钮 **"Code"** → **"Download ZIP"**
3. 解压到你喜欢的位置

## 第四步：初始化数据库

### Windows
1. 按下 `Win + R`，输入 `cmd` 打开命令行
2. 进入项目的 server/sql 目录（假设项目在 `D:\projects\pbl`）：
   ```cmd
   cd /d D:\projects\pbl\05_阶段三_后端开发\server\sql
   ```
3. 导入数据库（会提示输入密码，输入 `root`）：
   ```cmd
   mysql -u root -p < schema.sql
   ```
   没有报错就说明成功了 ✅

### Mac
打开终端，输入：
```bash
cd /你的路径/pbl/05_阶段三_后端开发/server/sql
mysql -u root -proot < schema.sql
```

## 第五步：启动后端服务

1. 打开命令行，进入 server 目录：
   ```cmd
   # Windows
   cd /d D:\projects\pbl\05_阶段三_后端开发\server

   # Mac
   cd /你的路径/pbl/05_阶段三_后端开发/server
   ```

2. 复制环境变量文件：
   ```cmd
   # Windows
   copy .env.example .env

   # Mac
   cp .env.example .env
   ```
   > ⚠️ 如果你安装 MySQL 时设置的密码不是 `root`，用记事本打开 `.env` 文件，把 `DB_PASS=root` 改成你的密码。

3. 安装依赖（只需要第一次运行，以后不用）：
   ```cmd
   npm install
   ```
   等待它跑完，看到 `added xxx packages` 就成功了。

4. 启动服务：
   ```cmd
   npm start
   ```
   看到 `🚀 Server running at http://localhost:3000` 和 `✅ MySQL connected`，说明后端启动成功了！🎉

5. **验证后端是否真的跑起来了**：打开浏览器，访问 http://localhost:3000/health
   - 如果看到 `{"status":"ok"}` 这样的 JSON，说明完美！
   - 如果打不开，回到前面检查 MySQL 密码对不对、3000 端口有没有被占用

> 💡 要停止服务，在命令行窗口按 `Ctrl + C`

## 第六步：启动微信小程序前端

1. 打开**微信开发者工具**
2. 点击 **"+"** 号（导入项目）
3. 项目目录选择：`你的项目路径\04_阶段三_前端开发\miniprogram`
   - 比如：`D:\projects\pbl\04_阶段三_前端开发\miniprogram`
4. AppID 那里，点击 **"测试号"**（不用自己申请，游客模式就能预览）
5. 项目名称随便填，比如"待办小程序"
6. 点击 **"确定"**
7. 项目会自动编译，等几秒钟就能看到登录页面了！

## 第七步：手机真机预览（可选）

1. 确保你的手机和电脑连在**同一个 WiFi**下
2. 在微信开发者工具顶部，点击 **"预览"** 按钮
3. 用微信扫描弹出的二维码
4. 手机上就能用了！

> ⚠️ 真机预览时需要修改后端地址为你电脑的局域网 IP：
> 1. 查看电脑 IP：Windows 在 cmd 输入 `ipconfig`，找 IPv4 地址（如 192.168.1.100）
> 2. 打开 `04_阶段三_前端开发/miniprogram/app.js`
> 3. 把 `apiBase: 'http://localhost:3000/api'` 改成 `apiBase: 'http://192.168.1.100:3000/api'`（换成你自己的 IP）
> 4. 在微信开发者工具中：详情 → 本地设置 → 勾选"不校验合法域名"
> 5. 重新点击预览

## 常见问题快速排查

### 问题1：npm install 报错
- **网络问题**：切换成手机热点再试，或者配置 npm 淘宝镜像：
  ```cmd
  npm config set registry https://registry.npmmirror.com
  ```
  然后重新 `npm install`
- **权限问题**：Windows 用管理员身份打开 cmd；Mac 命令前加 `sudo`

### 问题2：MySQL 连接失败（看到 ❌ MySQL connection failed）
- 检查 MySQL 服务是否启动：Windows 搜索"服务"，找到 MySQL 看状态是不是"正在运行"
- 检查密码：打开 `.env` 文件，确认 `DB_PASS` 是你设置的密码
- 检查端口：默认 3306，如果被占用，在 `.env` 里改 `DB_PORT`

### 问题3：小程序打不开 / 网络异常
- 确认后端已经启动了（浏览器能打开 http://localhost:3000/health）
- 微信开发者工具 → 详情 → 本地设置 → 确认 **"不校验合法域名、web-view...TLS版本..."** 是**勾选**的

### 问题4：端口 3000 被占用
```cmd
# Windows 查找并关闭占用 3000 端口的进程
netstat -ano | findstr :3000
taskkill /PID 查到的进程号 /F

# Mac
lsof -i :3000
kill -9 查到的PID
```
