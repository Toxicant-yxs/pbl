## 1. 引言

### 1.1 编写目的
本文档基于《PBL训练营实训计划_张桂莲.docx》九.20 课题的 7 项基础功能要求，细化"个人待办事项管理小程序"的功能与非功能需求，作为系统设计、编码实现与测试验收的依据。

### 1.2 项目背景
微信小程序已成为学生群体高频应用。现有待办 App 或功能臃肿、或体验割裂。本项目旨在通过一周的 PBL 训练营，完成一款轻量、聚焦、可视化的小程序待办工具。

### 1.3 术语定义
- **待办（Todo）**：用户登记的需要完成的事项，含标题、截止时间、优先级、分类等属性。
- **分类（Category）**：用户对待办的分组标签，如"学习"、"工作"、"生活"。
- **优先级（Priority）**：紧急程度，分为紧急 / 中等 / 较低 / 无。
- **完成率（Completion Rate）**：已完成数量 ÷ 总数量。

## 2. 项目概述

### 2.1 目标
为学生群体提供一款微信小程序形态的待办管理工具，支持快速记录、分类整理、截止提醒、数据统计。

### 2.2 用户角色

| 角色 | 描述 | 权限 |
|------|------|------|
| 普通用户 | 注册登录的小程序使用者 | 全部业务功能 |
| 系统管理员 | （本项目范围外，预留扩展） | 后续迭代 |

### 2.3 运行环境
- 客户端：微信小程序（基础库 ≥ 2.19.0，兼容 iOS / Android）
- 服务端：Node.js 18 LTS，Express 4.x
- 数据库：MySQL 8.0（亦可替换为 MongoDB）

### 3.3 核心用例详述

#### UC-01 用户注册
- **前置**：用户打开小程序首次进入
- **主流程**：
  1. 用户点击"立即注册"
  2. 输入用户名（4-20 字符，唯一）、密码（6-20 字符）、昵称
  3. 客户端校验后提交
  4. 后端 bcrypt 加密密码，插入 users 表
  5. 返回成功提示，自动跳转登录
- **异常**：用户名已存在 / 密码强度不足 / 网络异常

#### UC-04 新建待办
- **前置**：用户已登录
- **主流程**：
  1. 用户点击右下角 + FAB 按钮
  2. 进入"新建待办"页面，填写标题（必填）、描述、选择分类、优先级、截止时间
  3. 可选开启"提醒"开关并选择提醒时间
  4. 点击"保存"
  5. 客户端校验 → 调用 POST /api/todos
  6. 后端写入 todos 表
  7. 引导用户授权订阅消息（用于后续提醒）
  8. 返回成功，回到首页，列表刷新

#### UC-07 标记完成
- **主流程**：
  1. 用户在待办列表点击待办前的圆圈
  2. 乐观更新 UI，调用 PATCH /api/todos/:id/complete
  3. 后端更新 is_completed=1，completed_at=NOW()
  4. 连续打卡天数 +1

#### UC-15 数据统计
- **主流程**：
  1. 用户点击底部 Tab "统计"
  2. 并行调用 4 个统计接口
  3. 渲染完成率环形图、4 项核心指标、近 7 日柱状图

## 4. 非功能需求

### 4.1 性能需求
- 接口 P95 响应时间 ≤ 500ms
- 列表首屏渲染 ≤ 1.5s
- 核心接口响应 ≤ 3s
- 并发 ≥ 50 QPS

### 4.2 安全需求
- 密码 bcrypt 加密（saltRounds=10）
- 鉴权：JWT，HttpOnly 思路；小程序存 Storage
- 参数校验：express-validator
- SQL 注入：参数化查询
- XSS：前后端双重转义

### 4.3 兼容性需求
- 微信基础库 ≥ 2.19.0
- iOS 12+ / Android 7+

### 4.4 易用性需求
- 操作不超过 3 次到达核心功能
- 空状态、加载态、错误态全覆盖
- 关键操作（删除）二次确认

### 4.5 可维护性需求
- 后端遵循 MVC 分层
- 前端公共组件封装
- 接口版本化（/api/v1/...）
- 错误码统一规范

## 5. 异常场景清单

| 场景 | 表现 | 处理策略 |
|------|------|----------|
| 网络断开 | wx.request fail | Toast「网络异常，请检查」+ 保留本地数据 |
| Token 过期 | 401 | 自动跳登录页 |
| 必填项为空 | 客户端校验 | 红字提示具体字段 |
| 接口 5xx | 通用错误 | Toast「服务异常，请稍后再试」 |
| 订阅消息被拒 | 引导授权失败 | 降级为本地通知 + 提示 |
| 数据库异常 | 500 | 友好提示，详情见服务端日志 |
