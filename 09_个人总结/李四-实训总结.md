# 个人实训总结 · 李四（后端主程）

> 角色：后端主程  
> 负责：Express 服务、数据库、API 接口

## 一、做了什么

- 主导后端架构设计与 MySQL 表结构
- 21 个 RESTful 接口实现
- JWT 鉴权、bcrypt 加密、参数化查询
- node-cron 定时提醒任务
- 编写《API 接口文档》《数据库设计》

## 二、学到什么

- Express 中间件机制与 MVC 分层
- MySQL 索引设计与查询优化
- JWT 鉴权流程与无状态设计
- node-cron 定时任务
- API 文档先行的工程价值

## 三、踩过的坑

1. **MySQL TINYINT 返回数字**：前端 v-if 失效，DAO 层强转 boolean
2. **分页 total 不准确**：count 查询未复用 where 条件
3. **跨时区时间错位**：统一返回 ISO 8601 字符串
4. **数据库连接耗尽**：忘记 pool.release 导致并发崩溃

## 四、自我评价

- 后端开发：8/10
- 数据库设计：7/10
- 安全意识：7/10
